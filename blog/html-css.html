Classes vs ids

classes and ids in html are used to organize styling in css, classes allow styling to apply to any piece of code you tack the class on to, whereas ids are used to style an individual item. in the event I wanted all my paragraphs to look the same way, but maybe I wanted the text of one to be green I would use a paragraph class to standardize my webpage, and with the one paragraph an id would be used to give it special rules
classes and ids do very similar things but with different applications a good rule of thumb is to think about how often you will stylize something in that particular way, if you're only going to do it to a single element on a page, use an id, otherwise if the scale is large And you'll frequently be referring to that format, use a class.
